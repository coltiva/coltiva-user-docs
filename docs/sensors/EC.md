# EC

## Mounting

The EC sensor can be mounted using the sensor platform.

## Cleaning

Clean the sensor with a mild detergent, or isopropyl alcohol based disinfectant. If there's stubborn residue on the metal tips, use a scouring pad (e.g. Scotch-Brite) to remove it.
